## Portfolio Website

This is my protfolio website in which i show my projects, about myself and my resume 
in this website you can see and go through my projects like RigZen, EqVision, Car Washing Simualtor and
some more which i am developing or working around.

This website is build on Next.js framework 

you can download the app by:-

```bash
npx create-next-app@latest {your_folder_name}
```

To run locally use this command

```bash
npm run dev
```

To visit the website (local host):- http://localhost:3000/
